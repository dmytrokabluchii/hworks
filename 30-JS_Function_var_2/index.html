1. Напиши всі можливі варіанти створення функцій.

Variant of Function

1. Function Declaration (Объявление Функции)

Пример:
function sayHi() {
alert( "Привет" );
}

2. Function Expression (Функциональное Выражение)

Пример:
let sayHi = function() {
alert( "Привет" );
}

3. Функции-«колбэки» и анонимные функции

Пример:
function ask(question, yes, no) {
if (confirm(question)) yes()
else no();
}
ask("Вы согласны?",
function() { alert("Вы согласились."); },
function() { alert("Вы отменили выполнение."); }
);

Функции-«callback» - это функция переданная в качестве параметра другой функции! В примере функции объявляются прямо внутри вызова ask(...). У них нет имён, поэтому они называются анонимными. Такие функции недоступны снаружи ask (потому что они не присвоены переменным), но это как раз то, что нам нужно.

4. «Стрелочные функции» (arrow functions) =>

Стрелочные функции (arrow functions) позоляют сократить определение обычных функций. Стрелочные функции образуются с помощью знака стрелки =>, перед которым в скобках идут параметры функции, а после - собственно тело функции.

Пример:
let func = (arg1, arg2, ...argN) => expression;

Другими словами, это более короткий вариант такой записи:
let func = function(arg1, arg2, ...argN) {
return expression;
}

5. Рекурсия

Функцию можно также вызвать внутри самой себя в программировании называется рекурсией. Кроме этого необходимо предусмотреть условия для выхода из рекурсии!. Это не отдельный вид создания функции, но решил упомянуть и его все же!

Пример функция рекурсивного вычисления факториала:
function factorial(n) {
if ((n === 0) || (n === 1))
return 1;
else
return (n * factorial(n - 1));
}
